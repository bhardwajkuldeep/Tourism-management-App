Tourist Information System guides tourist which came from various regions. It provides brief detail about hotel, sightseeing, temple, parking, taxi-stand, ATM, educational institution, hospitals, garages, petrol pumps, road side stall etc. Till now a customer has to approach various agencies to find detail of places to book hotel, taxi. This often requires a lot of time and effort. So we are providing a platform in which a customer can find all these services integrated one application. The purposed system is a web based application and maintains a centralized repository of all related information. This system allows one to easily access the relevant information and make necessary travel arrangement. The users can decide about places they want to visit, find parking and get information for travel and accommodation. One can also get the information of garages, ATM, petrol pumps, hospital and some sightseeing spots. One can also get the location of road side stalls and can enjoy delicious fast food at low cost. User can easily access the location of various particulars like hotels, hospitals, petrol pumps, parking, taxi-stand, temple, educational institutions etc. So it become easily for the user to get the appropriate location with appropriate detail of every particulars.


Scope:-

• We provide the approach of various agencies in an integrated platform such as hotel, hospital, temple, sightseeing, taxi-stand ect.
• By using image mapping the user can remotely access the location of various places.
• User can easily get the location of the places with detailed information of those places.
• Detailed contact information for user so he or she can remotely access data or information.
• It provides user friendly interface.
